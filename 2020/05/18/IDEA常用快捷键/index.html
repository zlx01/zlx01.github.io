Intellij IDEA常用快捷键

发表于 2020-05-18 分类于 IDE 阅读次数：
本文字数： 869 阅读时长 ≈ 1 分钟

天下武功，唯快不破！

TAB 快速补全

ALT+ENTER 提示
ALT+INSERT 生成Getter、Setter等方法
ALT+F7 find usage
ALT+7 查看源码

SHIFT+F6 重构、重命名
SHIFT+F9 调试
SHIFT+F10 运行
SHIFT+ENTER 另起一行

CTRL+/ 行注释
CTRL+SHIFT+/ 块注释

CTRL+A 全选
CTRL+B 跳转到类、方法定义（同CTRL+CLICK）
CTRL+C 复制行
CTRL+D 复制并粘贴行
CTRL+E 打开最近文件
CTRL+F 查找
CTRL+SHIFT+F 在当前项目中查找
CTRL+G 跳转到某一行
CTRL+H 查看某一类的继承体系
CTRL+I 实现方法
CTRL+J 插入模板代码
CTRL+K commit
CTRL+SHIFT+K push
CTRL+L 搜索下一项（同F3）
CTRL+SHIFT+L 搜索上一项（同SHIFT+F3）
CTRL+M 滚动到文档中间位置
CTRL+N 查找类
CTRL+SHIFT+N 查找文件
CTRL+O 重载方法
CTRL+P 查看方法参数
CTRL+Q 类、方法文档
CTRL+R 查找并替换
CTRL+S 保存所有（一般都会自动保存）
CTRL+T pull
CTRL+SHIFT+T 测试类
CTRL+U 跳转到父类
CTRL+SHIFT+U 变大小写
CTRL+V 粘贴行
CTRL+SHIFT+V 选择复制历史粘贴
CTRL+W 选择代码
CTRL+SHIFT+W 选择代码
CTRL+X 剪切行
CTRL+Y 删除行
CTRL+Z 撤销
CTRL+SHIFT+Z 重做

CTRL+F2 停止运行
CTRL+F4 关闭标签页（同SHIFT+CLICK）
CTRL+F5 rerun
CTRL+F6 改变方法签名
CTRL+F7 find usage
CTRL+F8 打断点
CTRL+F9 build project
CTRL+F10 update running application
CTRL+F12 查看类结构

CTRL+ALT+L 格式化代码
CTRL+ALT+S 打开设置
CTRL+ALT+V 抽取变量
CTRL+ALT+M 抽取方法
CTRL+ALT+T 快速包裹代码

快速定位文件：点击任意文件，输入文件名
